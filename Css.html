ABOUT CSS

What is CSS?

Cascading Style Sheets (CSS) employ the words 'cascading' and style that defines how HTML code will look on a website depending on the data they wish to present. Without CSS, online sites are bland and uninteresting.


CSS is honored to Norwegian Hakon Wium Lie, who set out in 1994 to develop a universally compatible style sheet for the World Wide Web.


Lie's first test site using CSS was the Arena web browser. Lie went on to work collaboratively with Tim Berners-Lee and Robert Cailliau on CSS1, CSS2, and RFC 2318 versions. CSS became an established online standard in all of its requirements throughout its first decade of existence (1994–2004), greatly influencing the look and accessibility of the world wide web as we know it today.



Why is it important to learn CSS?

It’s easy to take a free Web template and build a website. But these templates rarely inspire by their elegance, so your website will look like every other site on the internet. By learning CSS you can modify these templates so that they display your colors and styles. Thus, you’ll have a customized website without a lot of effort.


CSS supports sites that look very different from page to page, without a lot of extensive coding. For example, many sites now perform slight color variations on the different sections of the site. Using page IDs, you can change the CSS for each section and use the same HTML structure for each section. The only thing that changes is the content and the CSS.


Many older websites that were built without CSS prove difficult to redesign. But once a site is built with CSS hooks, it can be redesigned very quickly. Changing things like the colors and backgrounds refresh how a site looks with very little effort. In fact, many sites now put up special versions of their sites for special occasions and they can do this because it only takes a few hours to create an alternate stylesheet for the occasion.

History of CSS

• CSS (1996) allows the user to select font style and size and change the colour of the text and background.


• CSS2 (1998) has capabilities that allows the user to design page layout.


• CSS3 (1999) allows the user to create presentations from documents and to select from a wider range of fonts including those from Google and Typecast. Uniquely, CSS3 allows the user to incorporate rounded borders and use multiple columns. CSS3 is considered to be easier to use (when compared to CSS2) because it has different modules
