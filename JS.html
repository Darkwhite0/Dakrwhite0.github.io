ABOUT JavaScript

What is JavaSript

JavaScript is a scripting language and one of the three main languages used to create websites. Whereas HTML and CSS provide structure and design to a website, JavaScript allows you to add functionality and behaviors to your website, allowing users to interact with the material in a variety of creative ways.


JavaScript is one of the powerful and flexible programming languages. It is one of the core technologies of World Wide Web. It is popular among developers as JS. The programming language is essential for developing web applications.



Why is it important to learn JavaScript?

JavaScript facilitates responsive design. As developers add more interaction and complexity to their programs, JavaScript has become an essential part of the Internet experience. It is necessary for search engines, ecommerce, content management systems, responsive design, social media, and mobile apps to exist.

History of JavaScript

• 1995- JavaScript was invented by Brendan Eich at Netscape and has since been updated to conform to ECMA-262 Edition 5 and later versions.

• 1997- Eich built JavaScript for Netscape Navigator, and it immediately became known as LiveScript. Another oddity in JavaScript history is that the team changed the name to JavaScript to match Netscape's support for Java in its browser.
